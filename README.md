# minesweeper
super duper minesweeper

## DEV
`npm run dev`で以下を実行  
- 更新に応じてサーバー再起動(port:3000)
- scssを圧縮しないでcssに変換

（デプロイ前）`npm run prod`で以下を実行
- scssを圧縮して変換

`npm start`はサーバー起動のみ. herokuが実行する
